구글시트 서버 설정 방법

1. 구글 스프레드시트 만들기
sheets.new 접속 → 새 시트 생성
하단 시트 탭에서 시트 3개 이름을 정확히: reservations, rooms, schedule

2. Apps Script 코드 입력
상단 메뉴 → 확장 프로그램 → Apps Script
기존 내용 전부 지우고 Code.gs 내용 붙여넣기
상단 SHEET_ID에 스프레드시트 URL의 /d/ 뒤 긴 문자열 입력
ADMIN_PW에 원하는 관리자 비밀번호 입력

3. 웹앱으로 배포
오른쪽 상단 배포 → 새 배포
유형: 웹앱 선택
실행 계정: 나(내 계정)
액세스 권한: 모든 사용자
→ 배포 클릭 → 승인 → .../exec로 끝나는 URL 복사

4. URL 입력
이 설정 화면의 Apps Script Web App URL에 복사한 URL 붙여넣기
서버 연동 활성화 체크 → 저장

⚠️ 코드를 수정할 때마다 새 배포가 아닌 배포 관리 → 편집으로 업데이트해야 URL이 유지됩니다.
